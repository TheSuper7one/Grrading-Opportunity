# Pokémon Grading Opportunity Dashboard

Streamlit app that pulls live Pokémon card pricing data and identifies potential grading opportunities.

## Features
- Live raw price data (Pokémon TCG API)
- Filter by set and raw price
- Designed to integrate PSA 9 / PSA 10 prices later

## Run Locally
```bash
pip install -r requirements.txt
streamlit run app.py
